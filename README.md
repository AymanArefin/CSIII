# CSIII - Computer Science III Portfolio

A collection of Java projects from Computer Science III, covering data structures, algorithms, game development with Mayflower, and AI implementations.

## Projects

| Project | Description |
|---------|-------------|
| [pokedemon-fizzbuzz](#pokedemon-fizzbuzz) | FizzBuzz variant (Poke/Pokemon) - basic control flow |
| [mayflower-intro-world](#mayflower-intro-world) | Introduction to Mayflower game engine - Cat, Dog, Yarn |
| [mayflower-score-display](#mayflower-score-display) | Mayflower world with score display |
| [mayflower-cat-ninja](#mayflower-cat-ninja) | Mayflower with Cat and NinjaGirl actors |
| [mayflower-multi-actors](#mayflower-multi-actors) | Mayflower with multiple character types |
| [mayflower-tile-world](#mayflower-tile-world) | Tile-based Mayflower world with ground and blocks |
| [mayflower-platformer](#mayflower-platformer) | Platformer game with gravity and animations |
| [custom-arraylist](#custom-arraylist) | Custom ArrayList implementation |
| [custom-linkedlist](#custom-linkedlist) | Custom generic LinkedList implementation |
| [map-interface](#map-interface) | Map data structure interface |
| [word-count](#word-count) | Word frequency counter using Map |
| [hashmap-implementation](#hashmap-implementation) | HashMap implementation with hash buckets |
| [tictactoe-gui](#tictactoe-gui) | Tic-Tac-Toe game with Mayflower GUI |
| [tictactoe-client](#tictactoe-client) | Tic-Tac-Toe network client |
| [tictactoe-client-alt](#tictactoe-client-alt) | Alternative Tic-Tac-Toe client |
| [royal-family-tree](#royal-family-tree) | Tree data structure - Royal Family genealogy |
| [tictactoe-minmax-ai](#tictactoe-minmax-ai) | Tic-Tac-Toe with MinMax AI and network server |
| [graph-implementation](#graph-implementation) | Graph and weighted graph implementations |
| [recursive-circles](#recursive-circles) | Recursive circles using Java AWT |
| [donkey-kong-platformer](#donkey-kong-platformer) | Full Donkey Kong-style platformer game |

## Requirements

- Java (JDK 8+)
- BlueJ (optional, for .bluej projects)
- Mayflower JAR (included for game projects)
